## 3. Video Explanation Script (3-5 Minutes)

### Title: Leaf Disease Detection System - CodeAlpha Internship Project

**[0:00-0:30] Introduction**
"Hi everyone! I'm excited to present my Data Science Internship project at CodeAlpha — a Leaf Disease Detection System using Deep Learning. Plant diseases cause up to 40% of global crop losses. My goal was to build an automated system that can classify leaf diseases from images with high accuracy, making agricultural disease diagnosis accessible to everyone."

**[0:30-1:00] Dataset Overview**
"I used the PlantVillage dataset, which contains 20,639 images across 15 classes — covering Tomato, Potato, and Pepper plants. The images are 256×256 JPEGs in RGB format. The dataset includes both healthy leaves and leaves affected by various diseases like Early Blight, Late Blight, Bacterial Spot, and Mosaic Virus."

**[1:00-1:30] Data Analysis & Preprocessing**
"First, I performed comprehensive exploratory data analysis — examining class distributions, image dimensions, and data quality. I then split the data 70/15/15 for training, validation, and testing. Data augmentation techniques like rotation, zoom, and brightness adjustment were applied to improve model generalization."

**[1:30-2:30] Model Development**
"I implemented four different architectures:

1. A Custom CNN with 4 convolutional blocks and dense layers
2. MobileNetV2 — lightweight and production-ready
3. EfficientNetB0 — state-of-the-art neural architecture search
4. ResNet50 — deep residual network

All models were trained with professional techniques including early stopping, learning rate scheduling, and model checkpointing."

**[2:30-3:30] Results & Evaluation**
"The results were outstanding! All four models exceeded 97% accuracy:

- EfficientNetB0 achieved the best performance at ~99% accuracy
- MobileNetV2 offered the best accuracy-to-speed ratio
- Even the Custom CNN from scratch reached ~97%

I created comprehensive evaluations including confusion matrices, ROC curves, training history plots, and error analysis to understand model behavior and limitations."

**[3:30-4:00] Inference System & Deployment**
"The system includes a production-ready prediction script that can classify new leaf images. Users simply run 'python predict.py --image leaf.jpg' and get the disease classification with confidence scores. The top-3 predictions are displayed for interpretability."

**[4:00-4:30] Conclusion**
"This project demonstrates that deep learning can provide accurate, scalable plant disease diagnosis. The complete codebase, including all models, training scripts, and documentation, is available on GitHub. Thank you for watching!"
